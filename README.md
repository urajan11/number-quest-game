
# 🔢 Number Quest — 5-Digit Guessing Game

A fun and interactive number guessing game built with HTML, CSS, and JavaScript. Also includes the original Java backend logic.

## 🎮 Live Demo

👉 [Play Now](https://urajan11.github.io/number-quest-game/)

---

## 📖 How to Play

- A random 5-digit number is generated by the system
- You have **5 attempts** to guess it
- After each guess you get feedback:
  - 🟢 **Green** — digit is correct and in the right position
  - 🟠 **Orange** — digit exists in the number but wrong position
  - ⚫ **Grey** — digit is not in the number at all
- Guess the number within 5 attempts to win!

- Animated starfield background
- Neon cyber theme

- Confetti animation on win
